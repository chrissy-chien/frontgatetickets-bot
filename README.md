# Running this bot

#### 1. Create a virtual environment

- On Windows:

```
py -m venv venv
venv\Scripts\activate
```

- On Linux/Mac:

```
python3 -m venv venv
source venv/bin/activate
```

#### 2. Install dependencies:

```
pip install -r requirements.txt
git clone https://github.com/obaskly/RecaptchaBypass.git
cd RecaptchaBypass
pip install -r requirements.txt
```
